This repository supports Math 103.1: Predictive Modeling for Text, a course for senior Applied Mathematics students on building, evaluating, and documenting predictive models for text data. The course combines machine learning foundations with practical workflows in Python, Jupyter notebooks, GitHub, and MLflow. Every modeling task is treated as a scientific experiment requiring reproducibility, traceability, ethical responsibility, and careful interpretation.

# math1031-predictive-modeling-for-text
Course materials for Math 103.1: Predictive Modeling for Text, emphasizing reproducible, traceable, and ethically responsible machine learning experiments with text data.
